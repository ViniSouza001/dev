Rede social de mapas mentais

1. Qual o objetivo?
	- O objetivo da rede social de mapas mentais seria todos os usuários terem sua própria conta, criar mapas mentais que ajudariam nos estudos tanto para si mesmos, quanto para os os outros, garantindo a comunicação e interação entre os usuários, filtros de buscas para mapas mentais, ferramentas para a construção de mapas mentais, funções como pesquisas, curtir e salvar os mapas mentais, comentários, entre outros. Os problemas que pretendemos resolver é ajudar a educação juntamente com a tecnologia lado a lado usando uma ferramenta poderosa que são as redes sociais. E será mais uma onde os estudantes poderão encontrar com facilidade mapas mentais das matérias ou assuntos que eles mais têm dificuldades ou dúvidas, e juntamente com os comentários, eles poderão fazer perguntas, questionamentos e sugestões para o autor do mapa mental.

2. Quem será o publico alvo?
	- O sistema desta rede social será direcionada especialmente para os estudantes em geral, mas não será limitado apenas a este público. Uma das principais intenções do projeto será a disponibilidade para qualquer público que deseja desfrutar de nossas ferramentas.

3. Quais as ferramentas que serão usadas?
	- As princpais ferramentas a serem usadas neste projeto serão:
		- Uso da API do Canva: O uso da API da ferramenta Canva é crucial, pois utilizaremos para o usuário poder criar um mapa mental dentro de nosso sistema, sem ter a necessidade de ter que intercalar entre os sistemas, um para criar e outro para salvar.
		- Banco de dados: O banco de dados será outra ferramenta de extrema importância, pois é o que irá salvar todos os documentos criados e todas as contas dos usuários, os itens curtidos, os comentários, os tópicos, e tudo que será salvo dentro do sistema

4. Como funcionará o sistema?
	- Basicamente, qualquer pessoa poderá entrar em nosso site e logar ou criar uma conta se não tiver uma. Podemos oferecer também para entrar com uma conta existente no Canva. Após ela estar logada, a página principal será os últimos mapas mentais feitos, tendo um limite de 20 demonstrações com tópicos sorteados de forma aleatória (isso na página principal). Nela o usuário pode navegar para a página de perfil onde estará todas as suas informações e se for a vontade, alterar alguma coisa. Poderá também navegar para a página de criar um novo mapa mental. Terá na navegação a opção para filtros de busca por tópicos, por exemplo as matérias que serão separadas, ou pela barra de pesquisa que o usuário poderá digitar as tags e dessa forma, aparecer os mapas mentais com as tags correspondentes.
	- Além disso, os comentários funcionarão de forma bem intuitiva e prática. O usuário terá que selecionar o mapa mental para ver com detalhes e as partes do comentário ficarão visíveis logo abaixo do rodapé do mapa mental.
	- No rodapé do mapa mental, ficará disponível ícones clicáveis para você compartilhar, salvar e curtir o mapa mental atual.
	- A função de compartilhar pode ser feita por um link, ou criar um pdf para você mandar para quem quiser, pode ser feito também um input para ser mandado por e-mail.

5. Quais os componentes terá em cada tela?
	- Um dos nossos objetivos é que todas as telas tenham uma interface limpa e amigável para os usuários. Para que todas as instruções sejam claras e diretas, fazendo com que não exista dúvidas para todos. Em cada tela, terá os seguintes componentes:

	- Tela de login: Essa será a primeira tela se o usuário não estiver logado. Será a tela em que terá menos componentes, pois apenas existirá um formulário para login, alguma imagem ou no máximo um GIF para uma interação com o usuário e uma boa primeira impressão. E não podemos esquecer do botão de criar conta, caso o usuário não tenha e queira criar uma, e talvez um botão para entrar como anônimo apenas para dar uma olhada na interface e as funções do sistema, mas nesse modo, algumas funções mais avançadas serão desabilitadas aos usuários no modo anônimo. Também terá as opções de entrar com outros sistemas, como Canva, Google, entre outros

	- Tela de cadastro: Será uma das telas com poucos conteúdos também. Terá o mesmo GIF ou imagem da tela de login, com um formulário para o usuário colocar seus dados, ou criar uma conta a partir de outros sistemas, como citado no item anterior.

	- Tela principal (home): A tela principal terá o cabeçalho com a logo do nosso projeto e uma navegação para as páginas de perfil e "mapas mentais". O conteúdo principal será a listagem dos últimos exemplos de mapas mentais, com tópicos sorteados e uma quantidade definida, por exemplo três mapas mentais de cada tópico. Dessa forma o usuário pode vizualizar uma porção de mapas mentais e escolher o que ele pretende olhar com detalhes.
	- Tela de perfil: Essa tela não haverá segredos. O usuário poderá olhar nela o seu nome de usuário (nickname), seu email, dados pessoais que pretendemos guardar como data de nascimentoe e se é estudante ou não, também terá opções de trocar senha
	- Tela "Mapas Mentais": Nesta tela, o conteúdo principal será a listagem de todos os tópicos de mapas mentais. No topo da página terá uma barra de pesquisa para o usuário digitar o conteúdo ou as tags que ele deseja buscar, e quando clicar em Buscar, será listado somente os mapas mentais que obtiverem as mesmas tags que foi pesquisado, ou somente os múltiplos/único tópico(s) que o usuário selecionou. No canto inferior direito, haverá um botão para o usuário criar um novo mapa mental, onde o usuário irá escolher a opção se ele deseja importar um mapa mental já criado do computador, onde será limitado o tipo de arquivo para ser aceito somente como .pdf ou outras extensões que forem possíveis, ou ele poderá escolher a opção de criar um mapa mental do zero pela ferramenta do Canva.

6. Quais os recursos a serem analisados e avaliados?
	- Disponibilidade da API pertencente ao Canva: este é um dos nossos principais recursos e diferenciais no projeto. Portanto, temos que avaliar todas as condições possíveis para usar a API da ferramenta Canva, avaliar também as documentações e disponibilidade para usar as ferramentas necessárias do mesmo.
	- Existência de um banco de dados gratuito: Como será um projeto de TCC e não haverá um retorno de renda, teremos que analisar os banco de dados que abrigarão todas as informações de maneira gratuita, sem cobrar taxas nem mensalidade, para que podemos apresentar de forma funcional e de forma completa. O que será deixado bem claro é que o banco de dados terá um espaço limitado para o número de usuários a serem logados e o número de mapas mentais a serem construídos, como o banco de dados será de versão gratuita

7. Como será a proteção de dados e a segurança?
	- Todas as senhas serão criptografadas com hash e os dados pessoais dos usuários não serão em nenhum momento compartilhado e mostrados em nenuma das páginas.
